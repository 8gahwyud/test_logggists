# Truvo Logists - Next.js приложение

Платформа для управления заказами логистов, перенесенная с ванильного JavaScript на Next.js.

## Структура проекта

```
truvo-logists/
├── app/                    # Next.js App Router
│   ├── layout.js          # Главный layout
│   ├── page.js            # Главная страница
│   └── globals.css        # Глобальные стили
├── components/            # React компоненты
│   ├── Header/            # Компонент заголовка
│   ├── BottomNav/         # Нижняя навигация
│   ├── OrdersPage/        # Страница заказов
│   ├── CreateOrderPage/   # Страница создания заказа
│   ├── ProfilePage/       # Страница профиля (с вкладками)
│   ├── OrderModal/        # Модальное окно заказа
│   ├── SubscriptionModal/ # Модальное окно подписки
│   ├── FinanceModal/      # Модальное окно финансов
│   └── SupportModal/      # Модальное окно поддержки
├── lib/                   # Утилиты и контекст
│   └── AppContext.js      # Глобальный контекст приложения
├── public/                # Статические файлы
│   └── img/              # Изображения
└── package.json          # Зависимости проекта
```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## API интеграция

Приложение использует Supabase для:
- Real-time обновления заказов
- Управления балансом
- Хранения данных пользователей

Конфигурация API находится в `lib/AppContext.js`

## Особенности

1. **Навигация профиля**: Вместо перерендера всей страницы используется навигация по вкладкам
2. **Модальные окна**: Все модальные окна открываются поверх контента без перезагрузки
3. **Real-time обновления**: Поддержка real-time обновлений через Supabase
4. **Адаптивный дизайн**: Сохранен оригинальный мобильный дизайн
